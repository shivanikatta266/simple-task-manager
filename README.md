# Simple Task Manager

## Description
A simple task manager web application built using HTML, CSS, and JavaScript.
The UI is implemented based on the provided mock-up.

## Features
- Add task (with title, description, and status)
- View tasks in To Do, In Progress, and Done
- Edit task
- Delete task
- Search tasks
- Dashboard with total task count
- Data persistence using browser localStorage

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository
2. Open index.html in any modern browser
3. The application runs locally without any setup

## Notes
- No backend is used
- Data is stored in browser localStorage
